# Android To-Do List App

A simple To-Do List Android application built using **Java in Android Studio**.

## Features

* Create task categories
* Add tasks inside each category
* Delete tasks easily
* Clean card-based user interface
* Data saved using SharedPreferences

## Technologies Used

* Java
* Android Studio
* RecyclerView
* SharedPreferences
* JSON for storing tasks

## How the App Works

Users can create headings such as:

* College
* School
* Shopping
* Travel

Inside each heading, multiple tasks can be added and managed easily.
